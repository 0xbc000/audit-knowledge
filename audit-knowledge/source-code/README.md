# Source Code

審計案例的原始碼，方便本地分析和參照。

## 目錄結構

```
source-code/
├── 2024-02-beanstalk/   # Beanstalk (算法穩定幣)
├── 2024-12-quantamm/    # QuantAMM (動態權重 AMM)
├── 2025-02-raac/        # RAAC (RWA + NFT Lending)
└── ...
```

## 命名規範

`{年份}-{月份}-{專案名}`

## 使用方式

案例研究文件在 `case-studies/` 目錄，這裡只存放原始碼。

對應關係：
- `case-studies/beanstalk-2024/` → `source-code/2024-02-beanstalk/`
- `case-studies/quantamm-2024/` → `source-code/2024-12-quantamm/`
- `case-studies/raac-2025/` → `source-code/2025-02-raac/`
